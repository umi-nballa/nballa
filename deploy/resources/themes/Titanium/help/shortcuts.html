Keyboard Shortcuts

If you see a menu, menu item, or button with an underlined letter in its name, you can select that item by holding down the Alt key and simultaneously pressing the letter that is underlined. For example, open the Policy menu by pressing Alt+P, and then choose the New Application menu item by pressing Alt+A.

You can sort on a second list column by holding the Shift key and simultaneously clicking the link in the column header.

You can also use the following additional keyboard shortcuts:

Press this To do this
Alt+Z Log out of PolicyCenter.

PageUp
PageDown

Home

 Jump to the beginning of a row in a list view table.
End Jump to the end of a row in a list view table.
